# Split You

A puzzle-platformer built in Unity where players create and control clones to solve synchronized environmental puzzles. Success requires planning, coordination, and clever use of multiple versions of yourself.

## Overview

In Split You, players can generate controllable clones and use them to overcome obstacles that would be impossible for a single character. Each level introduces new challenges that test timing, positioning, and strategic thinking.

The core gameplay revolves around coordinating multiple characters to activate switches and reach level objectives together.

## Features

- Clone-based puzzle mechanics
- Progressive level design
- Environmental challenges and obstacles
- Responsive platforming controls
- Multiple solutions and experimentation
- Built with Unity

## Controls

| Action | Key |
|----------|----------|
| Move Left | A / Left Arrow |
| Move Right | D / Right Arrow |
| Jump | Space |
| Spawn Clone | C |
| Reset Level | R |
| Pause/Menu | Esc |

> Controls may change as development continues.

## Installation

### Clone the Repository

```bash
git clone https://github.com/kdivyanshu1012/Split-You.git
```

### Open in Unity

1. Open Unity Hub
2. Click **Add Project**
3. Select the cloned repository folder
4. Open the project using the recommended Unity version
